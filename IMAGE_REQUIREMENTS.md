## Image Guidelines

### File Format
- Use **JPEG** for photos (.jpg)
- Use **WebP** for better compression (optional)
- Maximum file size: 200KB per image (compress using TinyPNG.com)

### Colors
- Look for images with **blue tones** (matches your brand color)
- Professional, clean, modern aesthetic
- Avoid overly saturated or busy images

### People in Images
- Diverse representation
- Professional attire
- Natural, authentic expressions
- Avoid stock photo "cheesy" poses

## How to Add Images

1. Download images from stock photo sites
2. Optimize/compress them at https://tinypng.com/
3. Rename files as specified above
4. Create folder: `public/images/`
5. Upload images to `/public/images/` in your repository
